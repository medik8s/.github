# Code of Conduct

The medik8s project has adopted the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the [full text](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) to understand what actions will and will not be tolerated.

## Reporting

If you experience or witness unacceptable behavior, please report it confidentially by contacting any of the maintainers listed in the repository's OWNERS file, or use the "Security and quality" tab on the repository to submit a private report.
